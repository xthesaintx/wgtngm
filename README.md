To help me track down and squash bugs as quickly as possible, please follow these steps when submitting a report.

- **Be Specific in the Title**: Try to summarise the issue in the title (e.g., "Asset Librarian: Icons not loading on Chrome"). Clear titles help us find and prioritise bugs much faster than generic ones like "Help" or "It's broken."

- **Tag the Module:** Always tag your post with the specific module (e.g., Campaign Codex, Mini Calendar).

- **Include Your Environment**: To replicate the bug, I need to know your setup. Please provide:
  - Module Version (e.g., v1.0.4)
  - Foundry VTT Version (e.g., v13 Build 327)
  - Browser/OS (e.g., Chrome on Windows 11, or Standalone App)

- **Description & Replication**: Please give a clear description of the behaviour and the steps needed to replicate it. While a video is great for context, a written description is still needed to help  walk through the logic.

- **One Bug Per Post**: Please keep each bug report to its own thread. This ensures that every issue can be tracked, discussed, and closed individually.

**Include Evidence:**
- **Screenshots/Video:** Visuals are always a huge help!
- **Console Errors:** Please include any errors or messages from the F12 Console (copy-pasting the text or a screenshot of the red error text is perfect).
